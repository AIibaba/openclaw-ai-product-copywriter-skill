*High-converting product copy for e-commerce listings, ad copy, landing pages, and brand messaging.*

A professional AI copywriting system that crafts persuasive, conversion-optimized product content. From Amazon listings and Shopify product pages to Facebook ads and email campaigns, generate copy that captures attention and drives sales.

## Features

| Feature | Description |
|---------|-------------|
| Product Listing Generator | Creates optimized titles, bullet points, and descriptions for Amazon, Shopify, and Etsy |
| Ad Copy Engine | Generates high-converting ad copy for Facebook, Google, and social media campaigns |
| Landing Page Writer | Crafts persuasive landing page copy with headlines, CTAs, and social proof sections |
| Brand Voice Builder | Develops and maintains a consistent brand voice across all content |
| A/B Variant Generator | Creates multiple copy variations for split testing and optimization |
| Multi-Language Support | Produces localized copy for international markets with cultural nuance |

## Getting Started

```bash
# Install via OpenClaw
openclaw install AIibaba/openclaw-ai-product-copywriter-skill
```

## Usage

```
write listing [product]    # Generate e-commerce product listing
write ad [product]         # Create ad copy for paid campaigns
write landing [product]    # Craft landing page copy
set voice [brand brief]    # Define your brand voice profile
variants [copy]            # Generate A/B test variations
```
